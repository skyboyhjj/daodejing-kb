# Five-Step Methodology — Complete Framework (v2.0 L1–L4 Layered)

> This document contains the full expanded methodology for the Five-Step Collaborative Reading Analysis. Version 2.0 adds the L1–L4 cognitive model and the 天道-人道-天人合一 dual-perspective framework.

## Part A: L1–L4 Cognitive Model

### Level Definitions

| Level | Label    | Target Audience                         | Core Need                                                        | Language Strategy                                    | Visual Cue                 |
| ----- | -------- | --------------------------------------- | ---------------------------------------------------------------- | ---------------------------------------------------- | -------------------------- |
| L1    | 大白话版 | Children (6–12), absolute beginners     | Intuitive first impression, zero intimidation                    | Life metaphors, conversational, zero jargon          | Green left border #a3c9b8  |
| L2    | 精读版   | Parents, interested learners            | Understand basic concepts, original text meaning                 | Define key terms, explain logical flow               | Gold left border #d4af37   |
| L3    | 应用版   | Practitioners, parents seeking guidance | Real-world application, case analysis                            | Multi-angle: management, psychology, personal growth | Teal left border #6b8e7a   |
| L4    | 学术版   | Scholars, deep enthusiasts              | Version variants, scholarly citations, cross-cultural comparison | Academic language, citations, philosophical rigor    | Purple left border #6a5e8a |

### L1 Content Creation Rules

**MUST**:
- Use everyday objects/scenarios as metaphors (water, garden, traffic, cooking, toys)
- Keep sentences short (max 15 Chinese characters)
- Start with "就像..." (Just like...) or "老子在这一章告诉我们..." (Laozi tells us in this chapter...)
- Include only ONE core insight per L1 block
- Use conversational connectors: "你看"/"你想啊"/"比如说"

**MUST NOT**:
- Use any philosophical terminology (道, 德, 无为, 玄, 自然 — use plain language equivalents)
- Include version variants or scholarly citations
- Exceed 3 sentences for any single point
- Use passive voice or academic connectors ("由此可见"/"综上所述")

**Example from Chapter 8**:
```
老子说，最高的善良就像水一样。你看水这东西多好啊——它让所有花草树木都活得好好的，
但从来不跟别人争功劳。哪儿低洼、哪儿没人愿意去，水就往哪儿流。老子告诉我们：做人也
要学水——不争不抢、甘心做别人不愿意做的事，反而最了不起。
```

### L2 Content Creation Rules

**MUST**:
- Define each core term on first use: "无为（不强行干预、顺其自然的态度）"
- Explain the chapter's logical flow: "这一章分为三个层次：首先...其次...最后..."
- Connect to at least one other familiar chapter
- Provide the original text alongside plain translation

**MUST NOT**:
- Dive into scholarly controversies (save for L4)
- Use untranslated classical Chinese
- Assume prior knowledge of Daoist terminology

### L3 Content Creation Rules

**MUST**:
- Anchor every insight in a recognizable modern scenario (workplace, family, personal dilemma)
- Provide at least two application domains: 个人修养 (personal) and one other
- Include concrete, actionable suggestions (not abstract "you should be more like water")
- Use real or realistic case examples

**MUST NOT**:
- Stay at the conceptual level without demonstrating application
- Use academic citation format — scenarios should be vivid, not footnote-heavy

### L4 Content Creation Rules

**MUST**:
- Compare at least two textual traditions (通行本 vs. 帛书本)
- Cite at least two commentators by name (王弼, 河上公, etc.)
- Include the 天道-人道-天人合一 structural analysis
- Engage with at least one modern scholarly perspective

**MUST NOT**:
- Repeat L2/L3 content in academic language
- Present one interpretation as definitively "correct" without acknowledging alternatives
- Introduce content irrelevant to this chapter's interpretation

## Part B: 天道-人道-天人合一 Dual Perspective

Every chapter analysis must address three interconnected dimensions:

### 天道 (The Dao Dimension)
What does this chapter reveal about Dao itself?
- Properties: What aspect of Dao's nature is illuminated? (emptiness, reversal, softness, non-contention, creative power, etc.)
- Cosmology: How does this chapter contribute to the Dao → De → Wanwu generative framework?
- Movement: What pattern of Dao's movement is demonstrated? (reversal, return, spontaneous emergence)

### 人道 (The Human Dimension)
What guidance does this chapter offer for human life?
- Self-cultivation: What inner work does this chapter prescribe? (stilling, emptying, softening, unifying)
- Conduct: What outward behavior is recommended? (non-contention, humility, knowing when to stop)
- Governance: If applicable, what political/leadership principle is advocated?

### 天人合一 (The Unity Dimension)
How do the Dao and human dimensions complete each other?
- Mirroring: How does human conduct mirror Dao's operation?
- Path: What is the specific "way" by which a human being aligns with Dao through this chapter's teaching?
- Culmination: What does the unified state look like? (玄德, 自然, 无为而无不为, etc.)

## Part C: Five-Step Methodology (unchanged from v1.0)

### Step 1: Text Close Reading & Semantic Anchoring

#### 1.1 Version Comparison (版本比对)
Compare at least two of three textual traditions: 通行本 (王弼本), 帛书本 (马王堆甲/乙), 竹简本 (郭店楚简).

#### 1.2 Character & Word Tracing (字词溯源)
For each core term, provide definition and at least one internal parallel from another chapter.

#### 1.3 Sentence Structure Analysis (句式辨析)
Identify patterns: parallelism, anaphora, metaphor, causal chains, rhetorical questions, conditional reasoning.

#### 1.4 Internal Evidence Priority (内证优先)
Map core concepts to their occurrences elsewhere in the Dao De Jing. Never cite external texts before exhausting internal evidence.

### Step 2: Context Construction & Philosophical Deduction

Three-tiered context: Micro (intra-chapter), Meso (inter-chapter thematic clusters), Macro (systemic positioning + Warring States intellectual history).

### Step 3: Controversy Manifestation & Interpretive Options

Active revelation → Option display (王弼, 河上公, modern) → Evidence weighing (textual support, logical coherence, systemic compatibility).

### Step 4: Logic Verification & Systemic Compatibility

Internal coherence check, external compatibility (庄子, 文子), cross-chapter verification table (min 5 rows), Occam's Razor application.

### Step 5: Meaning Transformation & Modern Application

Three-layer extraction (historical → universal → modern), three-domain transfer (personal, organizational, civilizational), parent-child activities (cognitive, experiential, dialogical), chapter golden quote.

## Part D: Application Mindset (四大心法)

1. **Suspend "Original Intent"** — pursue optimal interpretation, not historical certainty
2. **Distinguish "Statement" from "Construction"** — label epistemological level of each claim (L1 textual / L2 systemic / L3 cross-textual / L4 creative)
3. **Let the Classic Interpret Itself** — resolve ambiguities through internal cross-reference
4. **Respect Poetic Quality, Balance Logic** — never reduce metaphor to concept-definition
