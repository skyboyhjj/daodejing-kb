# L1–L4 Content Creation Guide

> This document provides detailed content creation guidance for each cognitive level, using Chapter 8 ("上善若水") as the complete worked example. Use this as the authoritative reference when generating layered content for any chapter.

---

## Level 1: 大白话版 (Beginner / Child)

### Writing Principles
- **Metaphor-first**: Open with "就像..." or "你看..."
- **Zero terminology**: Replace all philosophical terms with everyday language
- **Max 3 sentences per insight**: Children's attention spans demand brevity
- **Conversational tone**: Use "你想啊" / "比如说" / "老子告诉我们"
- **One core takeaway**: Each L1 block communicates exactly ONE idea

### Terminology Replacements (L1 Equivalents)

| Original Term   | L1 Equivalent                 |
| --------------- | ----------------------------- |
| 道 (Dao)        | 大自然的规律 / 宇宙的运行方式 |
| 德 (De)         | 好的品质 / 像道一样做事       |
| 无为 (Wuwei)    | 不瞎折腾 / 不强行改变         |
| 自然 (Ziran)    | 本来该有的样子                |
| 玄 (Xuan)       | 很深很深 / 说不清楚的         |
| 柔弱 (Rouruo)   | 软软的、温柔的                |
| 不争 (Buzheng)  | 不跟别人抢                    |
| 处下 (Chuxia)   | 愿意在低处 / 不争第一         |
| 圣人 (Shengren) | 最聪明的人 / 懂道理的人       |

### Chapter 8 Example — All Steps

**Step 1 (文本细读) L1:**
```
老子在这一章告诉我们：最好的善良，就像水一样。
水让花草树木、小鱼小虾都活得好好的，但它从来不嚷嚷"是我干的！"
哪儿低、哪儿脏、哪儿没人愿意去，水就往哪儿流。
老子说，如果你能像水一样——帮助别人但不抢功劳，愿意做别人看不上的事——那你就跟"道"差不多了！
```

**Step 2 (语境构建) L1:**
```
这一章是整本书里讲"怎样做人做事"的重要一章。老子的想法很特别：一般人都喜欢往高处走、喜欢被表扬，但老子说，学水的样子才对——甘心在低处，反而最有力量。
```

**Step 3 (争议辨析) L1:**
```
关于"水有七种好品德"这句话，学者们吵了很久——有的人说这段是后来加上去的，有的人说是老子自己写的。不管怎样，重要的是它告诉我们：善良是可以练出来的，就像水有七种好习惯一样，我们也可以一个一个地学。
```

**Step 4 (逻辑验证) L1:**
```
这一章说的道理，和书里其他地方说的完全一致。比如后面有一章说"天下最软的东西，能打败天下最硬的东西"——水就是最软的，但它能把石头滴穿。所以老子说的"不争"不是胆小，是另一种厉害。
```

**Step 5 (意义转化) L1:**
```
今天回家后，你可以做一个小实验：把一杯水倒进不同形状的碗里——看，水永远能找到最适合自己的形状。人也一样，不用硬碰硬，柔软一点、会变通一点，反而能去任何地方。
```

## Level 2: 精读版 (Learner / Parent)

### Writing Principles
- **Define on first use**: "无为（不强行干预、顺其自然的态度）"
- **Show logical flow**: "这一章分为三个层次..."
- **Connect chapters**: "这与第X章的...思想相呼应"
- **Original text + translation**: Always pair the classical Chinese with a clear modern rendering

### Chapter 8 Example — Steps 1, 2, 5

**Step 1 (文本细读) L2:**
```
本章的核心比喻是"水"——老子以水喻"道"在现象世界最完美的体现。水具有两种根本特性：
① "利万物而不争"——滋养一切而不争夺功劳（向外原则）；
② "处众人之所恶"——甘居众人厌恶的低处（向内原则）。

正因这两种特性，水"几于道"（接近于道）。随后老子以"七善"（居善地、心善渊、与善仁、言善信、正善治、事善能、动善时）将水的品德具体化为七项可实践的修养条目，最后归结为"夫唯不争，故无尤"的核心结论。
```

**Step 2 (语境构建) L2:**
```
本章在《道德经》全书中的位置：前有第1-7章奠定了"道"的本体论基础，本章是首次以自然意象（水）系统性阐释"道"的品格，为后续第66章（江海善下）、第78章（柔水攻坚）的水意象群奠定基础。

微观逻辑：水之"二性" → 七善展开 → 不争之核，构成"总—分—总"的经典论述结构。
```

**Step 5 (意义转化) L2:**
```
历史义：在战国争霸的时代背景下，老子以"水德"提出了一种与当时"力争上游"主流价值观完全相反的生命哲学。
普遍义：最高的善不表现为征服和控制，而表现为滋养和承载。
现代义：在竞争文化中找到"不争之争"的智慧——正如水，不争反而无所不至。
```

## Level 3: 应用版 (Practitioner)

### Writing Principles
- **Scenario-first**: "当你在工作中遇到...时"
- **Multi-domain**: Personal + organizational + civilizational
- **Concrete actions**: Not "be more like water" but "每周留2小时不安排任何事的'如水时间'"
- **Case examples**: Use recognizable modern situations

### Chapter 8 Example — Steps 3, 5

**Step 3 (争议辨析) L3:**
```
不同解读路径对实践的启示差异：
- 若采河上公"修身养生"路径 → "七善"是一套具体的身心修养方法，每日自省对照
- 若采王弼"哲学本体"路径 → "水"是对道性的体认，重在心态转变而非方法
- 实践建议：初学可从"七善"中每周选一项刻意练习，进阶后逐步内化为自然心态
```

**Step 5 (意义转化) L3 — 三维情境迁移:**

**个人修养：** 当你感到"付出不被看见"时——水的"善利万物而不争"提醒你：真正的价值在于利他行动本身，而非他人的认可。行动：本周做一件不求任何回报的好事，观察自己的内心感受。

**组织管理：** 服务型领导的本质正是"水德"。最优秀的管理者像水一样——为团队提供养分（资源、信任、空间），却把功劳归于团队。行动：下次团队成功时，尝试不说"我带领团队..."，而说"团队做到了..."。

**文明批判：** "处众人之所恶"在消费主义时代具有特殊意义——在一个鼓励"向上爬""争C位"的文化中，主动选择"处下"是一种精神上的自由宣言。行动：审视自己的消费和社交行为，哪些是真正需要的，哪些只是在"争位置"。

## Level 4: 学术版 (Researcher)

### Writing Principles
- **Version comparison**: Always compare 通行本 vs. 帛书本 for key variants
- **Named citations**: "王弼注云..."/"河上公注曰..."/"陈鼓应指出..."
- **Cross-cultural**: Engage comparative philosophy where relevant
- **天道-人道-天人合一**: Apply the dual-perspective framework explicitly

### Chapter 8 Example — All Steps (Integrated L4 Content)

**Step 1 (文本细读) L4 — 版本比对 + 字词溯源:**

版本比对：通行本"与善仁"→ 帛书本"予善天"。"仁"为儒家核心概念（《论语》109见），老子通篇对"仁"持批判立场（第18章"大道废，有仁义"；第19章"绝仁弃义"）——此处的"仁"若为原文，需以道家义重新定义（自然施予、不刻意为之）；若为后世窜入（帛书证据支持），原文当为"予善天"（给予善于顺应自然），更合全书体系。

字词溯源："几于道"之"几"字至关重要——河上公注为"近同"，王弼则审慎保留超越性。水虽为最佳喻体，但终究是现象界的存有，只能"近"道而不能"是"道。此"几"字维持了道不可言说的超越性边界。

**Step 2 (语境构建) L4 — 天道-人道-天人合一:**

天道维度：本章揭示"道"的运作呈现为"利而不争"与"处下不居"两种基本模式。水"几于道"意味着道的本质属性在现象界获得了最接近的感性显现。

人道维度：人效法水德的具体路径——"七善"体系从空间选择（居善地）到时间把握（动善时），覆盖了人的生存实践的全部维度。

天人合一维度：人通过效法水之天道（不争、处下、柔弱、应变），即可成就其人之道（七善归于不争，不争归于无尤）。"故几于道"的"故"字逻辑——正是因为水体现了道的属性，所以它接近道；正是人效法了水，所以人通过水接通了道。

**Step 3 (争议辨析) L4:**

争议一："与善仁"是儒家窜入还是老子原意？
- 窜入说（刘笑敢等）：帛书"予善天"为据，"仁"为后世儒者所改
- 重定义说（陈鼓应等）：老子可借用儒家术语而赋予新义，"仁"在此非儒家本义
- 权衡：帛书证据偏强，窜入说更合文本演变逻辑。但即便为窜入，"与善仁"已被两千年的解释传统接纳为水德内涵的一部分

争议二：七德是否为后世添加？
- 支持原作论：排比手法在《道德经》不罕见（ch12、ch41），且每句均有跨章互文呼应
- 支持添加论：七德的整齐度与全书散韵夹杂风格有别，"政善治"等条目政治化倾向明显
- 折衷说：核心思想属老子，当前整齐的四字表述可能经过后世整理

**Step 4 (逻辑验证) L4 — 跨文化比较:**

水的隐喻在中西哲学中的分野：
- 老子之水：柔弱、不争、处下、利万物 → 阴性智慧
- 赫拉克利特之水（"人不能两次踏入同一条河流"）：流动、变化、不可捉摸 → 变易本体
- 泰勒斯之水（"水是万物本原"）：质料因、始基 → 宇宙论本原
- 差异本质：老子以水喻道之"德"（价值品格），希腊哲学以水论道之"体"（物质本原）

**Step 5 (意义转化) L4:**

"水德"修行的阶段化模型（与儒家"修身次第"对照）：
- 水滴（初学）→ 溪流（渐进）→ 江河（贯通）→ 江海（化境）
- 此四阶段与《大学》"格物→致知→诚意→正心→修身→齐家→治国→平天下"的八条目形成有趣对照：儒家是外扩式的递进，道家是内敛式的深化

## Parent-Child Co-Reading Workflow

For each chapter, the parent and child follow this flow:

```
1. Parent reads L1 aloud to child        → Child understands the basic idea
2. Parent reads L2 silently              → Parent grasps concepts
3. Together, do the 亲子赋能活动          → Shared experiential learning
4. Parent optionally reads L3 later      → Deepen personal application
5. Parent optionally reads L4 later      → Scholarly enrichment
```

This workflow is explicitly designed so that **the parent always stays one level ahead of the child**, creating a natural "teaching by learning" dynamic.

## Quality Checklist for L1–L4 Content

Before finalizing any chapter, verify:

**L1 Checks:**
- [ ] Can a 10-year-old understand every sentence?
- [ ] Are all philosophical terms replaced with everyday language?
- [ ] Is there at least one concrete life metaphor?
- [ ] Is each L1 block 3 sentences or fewer?

**L2 Checks:**
- [ ] Are all core terms defined on first use?
- [ ] Is the chapter's logical flow explicitly stated?
- [ ] Is there at least one cross-chapter connection?

**L3 Checks:**
- [ ] Are there at least two application domains?
- [ ] Are suggestions concrete and actionable (not abstract)?
- [ ] Are scenarios recognizable and modern?

**L4 Checks:**
- [ ] Are at least two textual traditions compared?
- [ ] Are at least two commentators cited by name?
- [ ] Is the 天道-人道-天人合一 framework applied?
- [ ] Is there at least one cross-cultural or comparative reference?
