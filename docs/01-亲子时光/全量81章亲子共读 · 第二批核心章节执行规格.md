## 三、安全审核特别提示

基于我们在第 78 章的讨论，第二批审核中需特别关注以下安全事项：

### 3.1 新增通用安全提示

在 `family_metadata.json` 的 `safety_notes` 字段中，对包含以下潜在风险的章节增加标注：

| 潜在风险 | 涉及章节 | 安全提示 |
|:---|:---|:---|
| 战争隐喻 | 第46章（“天下有道，却走马以粪”） | “避免使用战争、兵马、征战等隐喻解释这一章。对于4-6岁，可以用‘吃饱了去种地’比喻和平；对于10-12岁，强调‘不贪心’而非批判战争。” |
| 道德误读 | 第24章（“企者不立”） | “当孩子描述‘骄傲’或‘炫耀’时，不做道德评判。用‘垫脚看远处站不稳’的比喻引导，不评判孩子的性格。” |
| 涉险比喻 | 第26章（“重为轻根”） | “避免用‘轻浮’评判孩子。4-6岁可以用‘大树根扎得深，风来了不怕’；7-12岁用‘做事要有准备’取代‘轻浮’。” |
| 社会批判 | 第62章（“道者善人之宝”） | “不将‘不善人’直接解释为‘坏人’。4-6岁说‘每个人都不一样’；7-12岁说‘道保护所有人’。” |

### 3.2 审核清单新增项

在所有章节的审核清单中，新增以下检查项：

- [ ] 是否存在将《道德经》概念（无为、返璞归真、不争等）与不守规则、不讲礼貌关联的表述？若存在，是否容易被误解为“不良行为=有德”？
- [ ] 对话模板中是否涉及对孩子的行为做道德评判？若存在，是否保持了客观描述而非价值判断？
- [ ] 当孩子描述犯错或不听话时，慧惠的回应是否保持了平常心和客观态度，而非借机说教或美化？
- [ ] 年龄段的互动设计中，是否避免了任何战争、军事、暴力等具象化比喻？

## 四、执行流程

### 第一步：运行 `scripts/extract_meta.py` 批量提取

```bash
cd scripts
python extract_meta.py --chapters "7,9,10,16,24,26,27,28,41,44,45,46,51,52,54,55,62,64,71,78"
```

提取完成后，自动在 `data/family_metadata_raw.json` 中生成初稿，`review_status` 统一标记为 `pending`。

### 第二步：逐章人工审核

使用本地 `admin/family-review.html` 审核控制台（若尚未部署，可先手动编辑 JSON 文件），逐章检查以下项目：

1. **core_idea** 是否符合该章原意，且适合亲子场景
2. **interaction_points** 中三个年龄段的引导方向是否符合认知水平
3. **parent_tips** 是否具体、可立刻操作
4. **safety_notes** 是否已标注本章特有的安全风险
5. 审核通过后将 `review_status` 改为 `approved`

### 第三步：缓存预热

审核通过后，运行 `scripts/warmup_cache.py` 预热新章节的对话缓存。

```bash
python scripts/warmup_cache.py --chapters "7,9,10,16,24,26,27,28,41,44,45,46,51,52,54,55,62,64,71,78"
```

### 第四步：更新 `data/family_metadata_review.md` 审核追踪表

记录每章的审核状态、审核人、审核日期和备注。
