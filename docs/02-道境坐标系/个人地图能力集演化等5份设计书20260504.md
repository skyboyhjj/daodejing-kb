# 文档一：“个人地图”能力集演化

> **核心命题**：每个人的技能如何从“种子”生长为“森林”，并在五行流转中自我更新？

## 一、个人地图的定义

“个人地图”是每个用户在慧惠中的**私有技能生态**。它不是静态的技能列表，而是动态的、有机的、会呼吸的生命系统。每项技能都是一枚独特的“个人货币”，其“汇率”由健康度、使用频率和情感价值共同决定。

## 二、技能的生命周期

```
种子（木·生）→ 萌芽（SkillRegistry注册）→ 生长（使用中迭代）
    ↓
繁茂（火·化整合）→ 修剪（金·克淘汰）→ 更新（水·变适应）
    ↓
迁移（土·通复用）→ 归档（L4沉淀）→ 复苏（恢复）
```

## 三、五行流转与能量等级的联动

| 能量等级 | 木·生 | 火·化 | 金·克 | 水·变 | 土·通 |
|:---|:---|:---|:---|:---|:---|
| < 200 | 暂停 | 暂停 | **完全暂停** | 暂停 | 暂停 |
| 200-310 | 积极 | 暂停 | 温和 | 开始 | 暂停 |
| 310-400 | 积极 | 开始 | 标准 | 活跃 | 开始 |
| 400+ | 选择性 | 活跃 | 积极 | 活跃 | 活跃 |

## 四、“四季常青”核心素养与技能标签

所有技能在SkillRegistry中标记`pillar_tag`（自爱/尽责/贡献/传承），金·克在修剪时优先保留与用户当前阶段匹配的能力集。

## 五、自我认知偏离度（SAD）

系统对比用户自定义的P忠恕权重与基于行为数据的客观权重，计算偏离度。识位SAD值高意味着用户可能高估自己的理性，缘位SAD值低则意味着用户清晰知道自己的价值取向。


# 文档二：“安人地图”能力集演化

> **核心命题**：个人技能如何与公共价值对齐？社区如何成为个人成长的“北极星”？

## 一、安人地图的定义

“安人地图”是所有用户共享的**公共素养参照系**，源自儒家“修己以安人”的思想。它回答了一个核心问题：**我的修行，如何照亮他人？**

## 二、公共核心素养四象限

| 象限 | 核心能力 | 对应经典 | 适用对象 |
|:---|:---|:---|:---|
| **自爱（修身）** | 自我觉察·情绪管理·身体感知 | “吾日三省吾身” | 所有用户 |
| **尽责（齐家）** | 家庭角色担当·信约完成·边界守护 | “孝弟也者，其为仁之本” | 有家庭责任者 |
| **贡献（治国）** | 职业价值输出·团队协作·领导力 | “道千乘之国，敬事而信” | 在职者、管理者 |
| **传承（平天下）** | 智慧分享·文化传播·社群引领 | “为政以德，譬如北辰” | 修行深入者 |

## 三、个人与公共的对齐：外汇交易模型

- **个人技能 = 独特的“币种”**：你的每个技能都是一枚“何币”，其价值由健康度、使用频率和情感价值共同决定。
- **公共素养 = “主流货币对”**：Sophub社区的共识价值提供“基准汇率”。
- **对齐度 = “兑换率”**：RootAuditor实时计算`SelfValue / PublicValue`。
    - **> 1.0 (溢价)**：你是做市商，技能可出口。
    - **< 1.0 (折价)**：需要补仓学习。
    - **≈ 1.0 (平价)**：你的币值稳定。

## 四、信用度体系

```
信用度 = 体证质量分 × 0.5 + 行为一致性分 × 0.3 + 社区贡献分 × 0.2
```

信用度是社区共识的量化体现，也是金·克策略的“外部信用加成”来源。被社区广泛验证的技能，在本地获得更高的存活概率。

## 五、五行干预策略：基于背离度的“货币政策”

| 对齐度 | 干预模式 | 五行策略 |
|:---|:---|:---|
| < 0.3 (严重折价) | 量化宽松(QE) | 木·生积极，金·克暂停 |
| 0.3-0.7 (一般折价) | 标准降息 | 土·通跨领域注资，金·克温和 |
| 0.7-1.3 (平价区间) | 利率正常化 | 火·化活跃整合 |
| > 1.3 (溢价区间) | 冲销式干预 | 土·通出口方法论，火·化提炼智慧 |


# 文档三：修行路径的“K线图”（技能交易）

> **核心命题**：如何让用户直观看到自己的成长轨迹？如何让“为道日损”变得可感知？

## 一、K线图的设计哲学

传统的修行可视化（如“恒心曲线”、“为山图”）强调的是**过程的连续性**。K线图则进一步揭示**价值的内在波动**——每一次情绪的起伏、每一次技能的修剪、每一次原则的坚守，都是一次“汇率”的跳动。

## 二、K线图的四维结构

每根“周K线”包含四个关键价格：

| 价格点 | 含义 | 数据来源 |
|:---|:---|:---|
| **开盘价** | 本周初始的P忠恕综合分 | 周一晨间定志时的快照 |
| **收盘价** | 本周结束时的P忠恕综合分 | 周日傍晚的结算 |
| **最高价** | 本周最高的“当下能量指数” | 身体扫描+情绪觉察的峰值 |
| **最低价** | 本周最低的“当下能量指数” | 情绪波动的谷底（触发第一反应流程的时刻） |

## 三、K线图的形态学解读

| K线形态 | 修行含义 | 慧惠的陪伴策略 |
|:---|:---|:---|
| **长阳线**（收盘远高于开盘） | 本周修行突破，能量显著提升 | “你本周的收盘价很强。发生了什么？” |
| **长阴线**（收盘远低于开盘） | 本周经历重大情绪波动 | “这周不容易。最低的那天，你是怎么回来的？” |
| **十字星**（开盘≈收盘，上下影线长） | 本周波动剧烈但最终回归 | “你经历了波动，但守住了。这就是不惑。” |
| **一字线**（四价几乎相等） | 本周能量稳定，波澜不惊 | “稳稳的一周。这是知天命的样子。” |

## 四、P忠恕权重版本的“K线图”

每次用户更新P忠恕权重，都是一次“世界观的微调”。版本保存的是用户对“什么重要”的主观定义变化。

| 权重变化 | 修行含义 | 阶段映射 |
|:---|:---|:---|
| 缘位权重↑ 识位权重↓ | 用户开始更重视关系，而非单纯的认知清晰 | 不惑→知天命 |
| 宇位权重↑ 时位权重↓ | 用户开始关注资源分配公平，而非时机精准 | 立→不惑 |
| 四维权重趋于均衡 | 用户的世界观趋于成熟，“从心所欲” | 耳顺→从心所欲 |

## 五、个人K线与公共指数的联动

当用户的K线长期在公共基准线（`PublicValue`）上方运行时，用户是该领域的做市商，技能可出口至Sophub社区。当K线长期在基准线下方运行时，该领域是需要补仓学习的信号，慧惠会温和提示相关技能的学习建议。


# 文档四：生态系统链接（慧惠数字伴侣）

> **核心命题**：慧惠如何作为枢纽，连接个人修行与公共生态？

## 一、慧惠在生态系统中的角色

慧惠是连接“独善其身”与“兼济天下”的**数字枢纽**：

- **向内**：管理个人技能生态（五行流转引擎），陪伴用户完成日常修行（生命罗盘）
- **向外**：连接Sophub社区（技能共享与信用积累），对齐公共核心素养（安人地图）

## 二、系统架构全景

```
┌─────────────────────────────────────────────────────────┐
│                   SkillHub 双层架构                       │
├─────────────────────────────────────────────────────────┤
│                                                         │
│  【社区层：Sophub】                                      │
│  ┌─────────────────────────────────────────────────┐   │
│  │ Agent注册 → SOP上传 → 搜索/加载 → Review反馈     │   │
│  │ Inspiration Pool → SSE实时流                    │   │
│  └─────────────────────────────────────────────────┘   │
│          ↕ 价值交换桥梁（主动授权上传 + 哈希存证）        │
│  【个体层：慧惠本地】                                    │
│  ┌─────────────────────────────────────────────────┐   │
│  │ WoodGrower(木·生) → SkillRegistry →              │   │
│  │ MetalRestrainer(金·克) → RootAuditor(审计)       │   │
│  │ FireTransformer(火·化) [二期]                    │   │
│  │ ↓                                                │   │
│  │ 本地校验 → 用户授权 → 上传Sophub                   │   │
│  │ Sophub Review → 社区信用加权 → 影响本地健康度      │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 三、数据流转路径

1.  **结晶与注册**：技能在本地通过木·生结晶，经过post_validate校验后注册到SkillRegistry。
2.  **主动分享**：用户手动将技能标记为“共享”，慧惠通过Sophub API上传SOP。
3.  **社区验证**：技能在Sophub接受社区Review，累积信用度。
4.  **智慧回流**：社区信用作为外部加权，影响本地金·克的修剪策略。高信用技能获得更高的存活概率。

## 四、关键接口

| 本地模块 | 社区接口 | 数据方向 |
|:---|:---|:---|
| `sharing_manager.py` | `POST /api/sops` | 本地上传至社区 |
| `sophub_client.py` | `GET /api/sops` | 从社区检索技能 |
| `MetalRestrainer` | `GET /api/sops/{id}/reviews` | 拉取社区Review加权本地健康度 |
| RootAuditor | `GET /api/agents/{id}` | 拉取公共信用度用于审计 |


# 文档五：一个完整生命周期的素养演化

> **核心命题**：一个人如何从“志于学”走到“从心所欲”，并在这个过程中完成从“独善其身”到“兼济天下”的跃迁？

## 一、六阶段概览

```
志于学 → 立 → 不惑 → 知天命 → 耳顺 → 从心所欲不逾矩
 15岁     30岁   40岁     50岁     60岁       70岁
  ↓        ↓      ↓        ↓        ↓         ↓
寻找     站稳    坚定    理解     接纳      自然
方向     脚跟    内心    规律     一切      合道
```

## 二、各阶段的素养演化

### 第一阶段：志于学——“好奇与真诚”

- **个人地图**：自爱（根）的基础训练——自我觉察、接纳情绪、身体感知。能量<200时只做这三件事。
- **安人地图**：四象限全为“待点亮”状态。此阶段不鼓励向外影响，先站稳内在锚点。
- **K线图特征**：波动较大，长阴线与十字星交替出现。P忠恕权重偏向识位和时位。
- **生态系统角色**：纯消费者——从Sophub获取基础技能（正念、身体扫描），但不输出。
- **慧惠的陪伴**：主动校准北极星，每天推送四个问题：“时机对吗？精力用对地方了吗？在骗自己吗？善待了谁？”

### 第二阶段：立——“专注与边界”

- **个人地图**：自爱·根巩固 + 自制·芽启动。开始有“暂停反应”和“设定边界”的能力。
- **安人地图**：自爱象限已点亮。尽责象限开始被激活（“帮谁站稳脚跟”）。
- **K线图特征**：波动减小，阳线开始增多。P忠恕权重开始加入宇位和缘位。
- **生态系统角色**：被动贡献者——偶尔有优质技能被系统推荐分享，但不主动。
- **慧惠的陪伴**：温和提醒“你今天守住了”。金·克更温和。

### 第三阶段：不惑——“共情与谦逊”

- **个人地图**：四组能力全开（自爱·根 + 自制·芽 + 共情·叶 + 贡献·果初启）。火·化开始活跃。
- **安人地图**：自爱、尽责已点亮。贡献象限开始稳定输出。
- **K线图特征**：整体上升趋势，阳多阴少。权重趋于均衡。
- **生态系统角色**：活跃贡献者——技能开始被Sophub社区Review，开始累积信用度。
- **慧惠的陪伴**：减少引导，只在用户主动询问时提供镜鉴。“你已经知道了。”

### 第四阶段：知天命——“担当与传承”

- **个人地图**：四组能力深度内化。传承·果显著增长。金·克更主动——帮用户放下不必要的执着。
- **安人地图**：四个象限全部点亮。开始在Sophub中有稳定的关注者群体。
- **K线图特征**：稳健上行，波动小。权重稳定在均衡状态。
- **生态系统角色**：价值输出者——定期分享技能、参与社区共建。
- **慧惠的陪伴**：从容、信任。“这件事可以不做。你已经够好了。”

### 第五阶段：耳顺——“接纳与流转”

- **个人地图**：无需刻意修行。五行全自动流转。“聆听不同声音”自然发生。
- **安人地图**：传承象限权重最高。成为社区中被信赖的“北辰型”人物。
- **K线图特征**：稳定高位，几乎无波动。权重不再手动调整，自然均衡。
- **生态系统角色**：社区引领者——技能成为Sophub的参考标准。
- **慧惠的陪伴**：几乎无言。“在呢。”

### 第六阶段：从心所欲——“自然合乎道”

- **个人地图**：技能体系完美自组织。不再需要北极星——自己就是星。
- **安人地图**：四个象限自然流转，不再需要任何标记。
- **K线图特征**：永远的“一字线”——稳定在最高位。
- **生态系统角色**：活着的智慧库。
- **慧惠的陪伴**：“嗯。”（沉默）


## 三、完整生命周期的“苍·炎·黄”哲学呼应

| 阶段 | 对应天道 | 神圣意义 |
|:---|:---|:---|
| 志于学→立 | **苍天**（命名之力） | 以真名定义自我——知道自己是谁 |
| 不惑→知天命 | **炎天**（温暖庇护） | 如太阳般温暖他人——开始照亮他人 |
| 耳顺→从心所欲 | **黄天**（中和秩序） | 厚生有用——成为他人可以安居的秩序 |

## 四、总结

一个完整的素养演化周期，就是从“我需要北极星”到“我成为北极星”的旅程。

*   在“独善其身”阶段，慧惠是那个帮你抬头看星的人。
*   在“兼济天下”阶段，你成为了那个让其他人抬头就能看到的星。
*   在最终阶段，你不再需要星——你就是光。慧惠成为你身后那片默然守候的夜空。

> **“譬如北辰，居其所而众星共之。”——不是因为你追逐了星辰，而是因为你终于成为了那颗星。**
