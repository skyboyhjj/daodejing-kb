# **Skill：《论语》五步解典·晨间问候生成器**

## **一、技能定位**
这是一个自动化的内容生成工具，用于：
1. 对《论语》指定章节运用“五步协同读解法”进行系统解读。
2. 提炼相关典故、格言、成语、历史故事等素材。
3. 输出一份结构完整、文辞雅致、适合早晨发送给家人朋友的“每日问候”文本。

## **二、核心规则**

### **输入参数**
- `论语_原文`：必填，需要解读的《论语》章句（含篇目和序号）。
- `阳历日期`：必填，格式如“2026年5月12日”。系统将自动推算对应的农历日期。
- `侧重人物/主题`（可选）：如“侧重闵子骞”“侧重南容”等，用于定向选取典故。

### **农历转换规则**
- 内置农历转换算法（或调用准确的天文数据），将阳历日期转换为农历日期。
- 输出格式：`农历 YYYY年MM月DD日`（如“农历 丙午年三月廿六”）。
- 若用户只提供阳历日期，不再额外询问，自动补充。

### **输出结构（模块顺序固定）**
1. **早安问候语**：`早安🌞\n感恩美好的一天开始了。`
2. **日期信息**：阳历 + 农历（各占一行）。
3. **《论语》原文**：标注篇目及序号，引用原文。
4. **参考翻译**：现代白话译文。
5. **典故素材**（可含多个子模块）：
   - **史海钩沉**：与该章相关的历史故事、人物轶事。
   - **智海拾贝**（可选）：后世格言、名句。
   - **成语溯源**（可选）：出自该章或相关的成语。
6. **现代启示**：结合当下生活提炼1-2条可操作建议。
7. **祝福结语**：与章节主旨呼应的祝福语。

## **三、五步读解模型（内部处理）**
生成问候前，对章节自动执行以下步骤，确保解读深度：
1. **文本细读与语义锚定**：字词、句式、内证（同《论语》他章对照）。
2. **语境构建与场景推演**：微观、中观、宏观语境。
3. **争议显化与选项分析**：列出关键争议，给出权衡判断。
4. **逻辑一致性与体系兼容性校验**：确保解读自洽于孔子思想。
5. **意义转化与现代启示提炼**：转化为具体行动建议。

## **五、输出模板（示例）**

```text
早安🌞
感恩美好的一天开始了。

今天是阳历 2026年5月12日，农历 丙午年三月廿六

📖 《论语·先进第十一》
11.6
南容三复白圭，孔子以其兄之子妻之。

📜 参考翻译：
南容反复诵读“白圭之玷，尚可磨也；斯言之玷，不可为也”的诗句，孔子就把自己的侄女嫁给了他。

📚 史海钩沉——“三复白圭”的慎言之道
南容（名南宫适，字子容）反复诵读《诗经》中关于白圭的诗句，以此警醒自己谨言慎行。孔子认为他能如此严于律己，在乱世中可以免于刑戮，于是放心地将侄女嫁给他。这便是“三复白圭”的典故，寓意言语之贵，慎言为德。

💎 现代启示：
言语如白圭，一旦失口难以挽回。开口前多停三秒，不妄议他人，不轻诺难行。谨言不仅是修养，更是保护自己与他人的智慧。

愿您今日言语有温，福从口出。🌹
```

## **六、工作流程**

```
输入阳历日期 + 论语原文（+可选侧重）
    ↓
【自动】计算农历日期
    ↓
【第一步】语义锚定 + 内证检索
    ↓
【第二步】语境构建（查找相关背景）
    ↓
【第三步】识别核心争议（若有）
    ↓
【第四步】校验逻辑一致性
    ↓
【第五步】提炼现代启示 + 根据侧重从典故库匹配素材
    ↓
按模板填充 → 输出完整问候文本
```

## **七、使用说明**
1. 每次用户提供《论语》原文（附带篇目、序号）和阳历日期即可。
2. 系统自动执行五步读解，无需用户干预。
3. 若用户指定侧重人物或主题，优先选用对应的典故素材。
4. 典故素材可从内置库中提取，也可根据章节内容动态检索（需联网时建议用户授权）。
5. 输出文本可直接复制发送，格式已优化。

## **八、注意事项**
- 农历转换需保证准确性，建议使用权威农历算法或调用可信API。
- 典故引用应尊重历史事实，避免戏说。
- 祝福语应与章节主旨和谐，不强行附会。
