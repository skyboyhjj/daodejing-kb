## 一、任务概述

在 5 月 12 日汶川大地震 18 周年纪念日，网站首页和慧惠聊天窗口自动展示安静的纪念内容。**仅当日有效，用户可关闭，不强制。**

### 核心交付：

1. **首页纪念卡片**（方案 A）— 动态注入的轻量弹窗，淡墨色基调，当日显示，可关闭
2. **慧惠聊天纪念欢迎语**（方案 C）— 聊天窗口打开时展示，包含《道德经》原文引用


## 二、操作文件

| 文件 | 操作 | 说明 |
|:---|:---|:---|
| `public/js/memorial-512.js` | **新建** | 纪念功能独立 JS，不污染聊天组件代码 |
| `public/css/memorial-512.css` | **新建** | 纪念卡片独立样式 |
| `public/index.html` | **修改** | 引入以上两个新文件 |

**设计原则**：纪念逻辑完全独立于聊天组件，方便 5 月 13 日以后直接删除相关文件，不留任何残留代码。


## 三、日期判断逻辑

**JavaScript `getMonth()` 返回 0 基索引**：0 = 1月，1 = 2月，...，**4 = 5月**。

```javascript
var today = new Date();
if (today.getFullYear() === 2026 && today.getMonth() === 4 && today.getDate() === 12) {
    // 显示纪念内容
}
```

5 月 13 日此条件自动失效，所有纪念内容不再出现。


## 四、方案 A：首页纪念卡片

### 4.1 触发条件

仅在 2026-05-12 当天触发，且用户未关闭过（localStorage 标记 `huihui_memorial_512_dismissed` 不为 `true`）。

### 4.2 卡片内容

```
🕊️ 今天，我们安静地记得

十八年前的今天，汶川大地震。

《道德经》说：
“飘风不终朝，骤雨不终日。”

再大的风也会停，再大的雨也会歇。
而活着的我们，继续前行——
带着对逝者的怀念，
和对生命的郑重。

[开始今天的阅读]  [安静地关掉]
```

### 4.3 卡片样式

- 居中弹窗，半透明深色遮罩背景（`rgba(0, 0, 0, 0.3)`）
- 卡片为宣纸白底色（`#faf8f0`），淡墨绿文字（`#2e5f5f`）
- 最大宽度 420px，圆角 12px，阴影柔和，内边距 32px
- 行间距 1.8，字体 15px
- 加载后约 1 秒以淡入动画出现（fade in, 0.6s）
- 关闭时带淡出动画（fade out, 0.4s）
- 点击遮罩区域不关闭卡片（防止误触）
- 移动端卡片占 90% 宽度，字号缩小至 14px

### 4.4 交互逻辑

- 点击“开始今天的阅读” → 关闭卡片，平滑滚动至首页主内容区，写入 localStorage
- 点击“安静地关掉” → 关闭卡片，写入 localStorage
- localStorage 标记仅在当天有效，5 月 13 日不再读取


## 五、方案 C：慧惠聊天纪念欢迎语

### 5.1 触发条件

仅在 2026-05-12 当天，且聊天窗口的欢迎语可被覆盖时触发。**不受卡片关闭影响**——即使用户关闭了首页卡片，打开聊天窗口时仍显示纪念欢迎语。

### 5.2 欢迎语内容

```
你好呀！我是慧惠。

今天是5月12日。十八年前的今天，汶川大地震。

老子说：“飘风不终朝，骤雨不终日。”
再大的风雨都会过去，而活着的我们，
带着对生命的郑重，继续前行。

今天，你想从《道德经》的哪一章开始？
```

### 5.3 实现方式

在 `memorial-512.js` 中覆盖聊天窗口的默认欢迎语：

```javascript
var MEMORIAL_WELCOME = '你好呀！我是慧惠。\n\n' +
    '今天是5月12日。十八年前的今天，汶川大地震。\n\n' +
    '老子说：“飘风不终朝，骤雨不终日。”\n' +
    '再大的风雨都会过去，而活着的我们，\n' +
    '带着对生命的郑重，继续前行。\n\n' +
    '今天，你想从《道德经》的哪一章开始？';

window.HUIHUI_WELCOME_OVERRIDE = MEMORIAL_WELCOME;
```

`huihui-chat.js` 中需配合增加一行判断（若尚未支持，本次为其增加 3 行代码——在设置欢迎语时优先读取 `window.HUIHUI_WELCOME_OVERRIDE`，存在则使用覆盖版本，不存在则使用默认版本）。
