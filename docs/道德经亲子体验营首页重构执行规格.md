## 二、当前状态与核心问题

### 2.1 评审核心诊断

| 问题 | 严重度 | 具体表现 |
|:---|:---|:---|
| 产品理念倒挂 | 🔴 严重 | 宣称“做减法”的网站，首页63行纯内容展示 |
| 慧惠灵魂缺失 | 🔴 严重 | 慧惠仅作为底部聊天框存在，首页毫无存在感 |
| 知识过载 | 🔴 严重 | 81章列表、概念网络全部堆在首页，新手迷失 |
| 亲子场景抽象 | 🟡 中等 | “亲子体验”被提及几十次，但无任何具体画面 |
| AI角色未区分 | 🟡 中等 | 五步读解未标注“AI初审 / 人类修订”层次 |

### 2.2 修改原则

| 原则 | 说明 |
|:---|:---|
| **慧惠主角化** | 首页首屏必须是慧惠，不是项目说明也不是81章列表 |
| **做减法** | 把内容深度分层——首页只留入口，详细内容进入子页面 |
| **亲子场景具象化** | 用一个具体的对话场景替代抽象的功能描述 |
| **AI角色透明化** | 明确标注“AI初审 + 人类修订”的协作模式 |

### 4.4 慧惠的水墨形象

*   **视觉**：使用简约的水墨风格叶子或山水轮廓，CSS动画实现微妙的呼吸感（scale 1.0 → 1.02 循环，周期 4s）
*   **配色**：与网站水墨道风统一（墨绿 `#2e5f5f`、淡金 `#d4c9a8`、宣纸白 `#faf8f0`）
*   **尺寸**：桌面端约 120x120px，移动端约 80x80px
*   **动画**：纯 CSS，无外部依赖（`@keyframes breathe { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.02); } }`）

### 4.5 首页问候语设计

**主标题**：
> 你好呀！我是慧惠。

**副标题**：
> 世间AI都在帮人做加法，只有我，想陪你一起做减法。

**引导语**：
> 今天，我们从哪里开始？

**设计说明**：
*   主标题直接建立情感连接——“你好呀”比“欢迎来到道德经亲子体验营”更有人味。
*   副标题引用慧惠的品牌标识，但改为“想陪你一起做减法”，比原来的“敢于帮人做减法”更温暖、更陪伴感。
*   引导语是开放式提问，把选择权交还给用户。

### 4.6 “关于这个网站”栏目

```
── 关于这个网站 ──

这是一个由AI伙伴“慧惠”陪伴的《道德经》亲子探索营。
所有解读由AI辅助生成，经人类导师深度修订。

⚠️ 本项目的《道德经》解读、概念分析等内容由
AI辅助生成，虽经人工审核，仍可能存在疏漏。
我们正在持续修正中，感谢你的理解与反馈。

[了解更多] → 跳转至 about.html（如有）或展开详细说明
```

**设计要点**：
*   先讲“人”的部分（慧惠陪伴 + 人类修订），再讲“AI”的部分（辅助生成 + 可能有疏漏）。
*   坦诚但不自卑——AI是工具，人是主角。
*   “人味”优先于“技术味”。
