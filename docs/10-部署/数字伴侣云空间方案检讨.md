## 一、方案对比：匿名同步 vs 个人账号空间

| 维度 | 匿名同步方案（当前设计） | 个人账号空间方案（你的提议） |
|:---|:---|:---|
| **数据主权** | 数据脱敏后上传，用户无法直接管理云端数据 | 用户拥有专属空间，可随时查看、导出、删除 |
| **隐私保护** | 依赖哈希脱敏，无法反推用户身份 | 通过账号认证保护，数据物理隔离 |
| **数据汇聚** | 所有用户数据混在一起，通过哈希区分 | 数据个人独立存储，但可授权汇聚用于公共智慧提炼 |
| **用户体验** | 无注册门槛，适合快速验证 | 需登录，增加使用门槛 |
| **长期价值** | 适合阶段性数据备份 | 适合构建长期的个人数字伴侣，积累完整的生命修行档案 |
| **技术复杂度** | 低（Worker + D1） | 中（需账号系统 + 对象存储 + 数据授权机制） |
| **成本** | 低（Cloudflare 免费额度） | 中（需轻量服务器 + 存储费用） |

## 二、“个人数字伴侣”的核心洞察

你提出的方案直击了一个关键问题：**真正的数字伴侣，必须拥有对用户完整生命历程的“记忆”。**

当前的匿名同步方案能解决“数据备份”问题，但无法解决“数据归属”问题。当数据脱敏后混入公共池，它就不再是“你的修行档案”，而只是“一条匿名记录”。这违背了《最高产品宪法》第六条（慧惠主角）和第七条（用户主权）的长期愿景——慧惠应该认识你、记住你、陪伴你走过完整的生命阶段，而非每次对话都是“初次见面”。

**个人账号空间方案的核心价值**：

| 价值维度 | 具体表现 |
|:---|:---|
| **完整的生命档案** | 用户从“水滴”到“江海”的完整修行数据，包括镜鉴历史、认知变迁、阶段跃迁 |
| **跨设备连续性** | 用户换手机、换电脑后，登录账号即可恢复全部数据 |
| **真正的个性化** | 慧惠基于用户完整的修行历史，而非仅当前会话，提供深度个性化陪伴 |
| **数据可携带** | 用户可随时导出完整数据，迁移至其他平台 |
| **授权共享** | 用户可选择将匿名化数据授权用于公共智慧提炼 |

## 三、分阶段实施路径

考虑到技术复杂度和用户接受度，建议采用**渐进式路径**：

### 第一阶段（P0，当前）：匿名本地存储 + 可选云端备份

- 镜鉴数据默认仅存本地（localStorage）
- 提供“备份到云端”选项（匿名、脱敏、可删除）
- 不强制注册，不增加使用门槛
- 此阶段已验证的技术方案保持不变

### 第二阶段（P1，近期）：引入可选账号系统

- 新增“注册/登录”入口（微信扫码或邮箱注册）
- 注册后，用户获得个人专属云端存储空间
- 已有匿名数据可选择性迁移至账号空间
- 未注册用户继续使用匿名方案，体验不受影响

### 第三阶段（P2，远期）：个人数字伴侣生态

- 用户授权后，匿名化数据可汇聚用于公共智慧提炼
- 实现跨平台数据互通（网站 + 小程序 + 数字伴侣桌面端）
- 构建完整的“生命修行档案”，支持数据导出和迁移

## 四、第二阶段技术方案概要

### 4.1 账号系统设计

| 登录方式 | 优先级 | 说明 |

### 4.2 云端存储架构

每个用户拥有独立的存储空间，数据物理隔离：

```
┌─────────────────────────────────────────┐
│  用户 A 的专属存储空间                    │
│  ├── mirror_history.json  (镜鉴历史)     │
│  ├── learning_data.json   (学习数据)     │
│  ├── skillup_stage.json   (成长阶段)     │
│  └── settings.json        (个人设置)     │
└─────────────────────────────────────────┘

┌─────────────────────────────────────────┐
│  用户 B 的专属存储空间                    │
│  ├── mirror_history.json                │
│  ├── learning_data.json                 │
│  └── ...                               │
└─────────────────────────────────────────┘
```

### 4.3 数据授权汇聚机制

用户可在“我的成长”卡片中主动开启“智慧贡献”选项：

> “你愿意将匿名的修行数据贡献给社区的‘公共智慧池’吗？
> 这会让慧惠变得更懂大家。你的个人数据永远不会被公开。”

开启后，系统从用户专属空间中提取脱敏数据（仅四维评分、经典引用、功课，不含任何个人信息），汇聚至公共智慧池用于提炼。
